# Changelog

Upgrade to the latest version:

=== "Linux/macOS"

    ```
    $ pip3 install -U apiflask
    ```

=== "Windows"

    ```
    > pip install -U apiflask
    ```

or check the currently installed version first:

=== "Linux/macOS"

    ```
    $ pip3 show apiflask
    ```

=== "Windows"

    ```
    > pip show apiflask
    ```
